void CInstanceConfigurableElement::getListOfElementsWithMapping ( std::list< const CConfigurableElement * > &  configurableElementPath) const
virtual

Get the list of all the ancestors that have a mapping.

The mapping is represented as a std::string of all the mapping data (key:value) defined in the context of the element. In this class, the method is generic and calls its parent getListOfElementsWithMappings(...) method.

Parameters
in:out]configurableElementPath List of all the ConfigurableElements found that have a mapping. Elements are added at the end of the list, so the root Element will be the last one.

bool CInstanceConfigurableElement::sync ( CParameterAccessContext &  parameterAccessContext) const
protected

Performs the sync if the AutoSync is enabled.

If AutoSync is disabled, any call to sync will returns true, even if synchronization has not been done. It will happen when the AutoSync will be switched back on.

Parameters
in:out]parameterAccessContext Parameter access context object
Returns
true if the synchronization succeded or if the AutoSync is off, false otherwise.
