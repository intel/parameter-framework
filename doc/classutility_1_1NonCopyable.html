Detailed Description

Base class for private inheritance to make the derived class non copyable.

Sometime implementing copy/move semantic is complicated or not useful. The traditional way is to declare the relevant methods private in c++ < 11 or deleted in c++ > 11.

It is easier and self documented to derive from this class.

For more documentation see: http://www.boost.org/doc/libs/release/libs/utility/utility.htm#Class_noncopyable
