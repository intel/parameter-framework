Detailed Description

Class that gather errors during serialization.

Provided with an initial empty buffer (strError), an instance of this class can describe an error.

This error will be accessible formated in the aforementioned buffer after destruction. Ie. the provided buffer (strError) is in an undefined state between construction and destruction and should not be accessed in between.
